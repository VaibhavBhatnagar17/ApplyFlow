# JobPilot

AI-powered job search dashboard. Upload your resume, set your preferences, and get a personalized dashboard with matched jobs, cover letter generation, company research, and application tracking.

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Features

- **Resume Parsing** — Upload a PDF, auto-extract skills
- **Job Dashboard** — 146+ curated listings scored against your profile
- **Company Database** — Browse 250+ companies with career links
- **Live Job Search** — Search LinkedIn, Indeed, Naukri, Google Jobs
- **Cover Letters** — One-click tailored cover letter per job
- **Application Tracker** — Track status across your pipeline

## Deploy

1. Push to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect repo, pick `app.py`, deploy
